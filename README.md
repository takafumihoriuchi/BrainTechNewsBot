# BrainTechNewsBot

世界の最先端 BrainTech / BCI（脳コンピュータインタフェース） / 神経科学のニュースを自動収集し、Gemini AI で専門的な日本語解説付きの要約を生成して Bluesky に配信するインテリジェンス・ボットです。

## 概要

このプロジェクトは、情報の更新速度が極めて速いニューロテクノロジー分野において、グローバルな一次情報をいち早くキャッチアップし、構造化されたインサイトとして共有することを目的としています。単なる翻訳ではなく、AI が専門的な前提知識を補足し、可読性と情報密度を両立させたフォーマットで配信します。

## 主な機能

- **マルチソース・アグリゲーション**: 厳選された 7 つの主要専門メディアから RSS フィードを監視し、広範囲な情報を網羅。
- **インテリジェント・フィルタリング**:
  - **キーワード優先ロジック**: Neuralink, Synchron, Stentrode 等の特定キーワードを含む重要トピックを最優先でピックアップ。
  - **履歴管理**: 直近 100 件の投稿 URL 履歴を保持し、二重投稿を確実に防止。
- **AI 構造化要約（Gemini API）**:
  - 専門用語を噛み砕き、体言止めを避けた自然な文章形式での 3 点要約。
  - 出版日 `(YYYY/MM/DD)` を見出しに付与し、情報の新鮮さを可視化。
  - 300 文字の制限内で情報密度を最大化し、全行の間に空行を挿入することで高い可読性を実現。
- **リッチな投稿（Bluesky API）**: 記事のアイキャッチ画像やタイトルを含めたリンクカード（Embed）形式での投稿。
- **高頻度自動更新**: GitHub Actions を利用し、1 時間おきに完全自動で稼働。

## ニュースソース（RSS）

現在、以下の 6 つの信頼できるソースから情報を取得しています。

- **Nature (BMI)**: 世界最高峰の学術誌による脳マシンインタフェース報告
- **ScienceDaily**: 論文ベースの研究成果（BCI カテゴリ）
- **Neuroscience News**: 神経科学全般の最新トレンド
- **MIT News**: マサチューセッツ工科大学による技術革新
- **IEEE Spectrum**: 工学・ハードウェア視点のデバイスニュース
- **EurekAlert!**: 世界中の研究機関による公式プレスリリース

## 技術スタック

- **Language**: Python 3.12
- **AI**: Google Gemini API (`gemini-1.5-flash`)
- **Infrastructure**: GitHub Actions
- **Library**:
  - `atproto`: Bluesky API 操作
  - `feedparser`: RSS 解析
  - `beautifulsoup4` / `requests`: OGP 情報の取得

## 投稿フォーマット

プロフェッショナルな情報伝達のため、以下の構造化ルールを採用しています。

1. **【タイトル要約】（YYYY/MM/DD）**
   (空行)
2. **要点1（■）**: 文章形式での解説（句点なし）
   (空行)
3. **要点2（■）**: 文章形式での解説（句点なし）
   (空行)
4. **要点3（■）**: 文章形式での解説（句点なし）
   (空行)
5. **専門的補足（※）**: 理解を助ける前提知識

## セットアップ

### 環境変数の設定

リポジトリの `Settings > Secrets and variables > Actions` に以下の秘密鍵を設定してください。

| 変数名 | 内容 |
| :--- | :--- |
| `GEMINI_API_KEY` | Google AI Studio で発行した API キー |
| `BLUESKY_HANDLE` | Bluesky のハンドル名（例: user.bsky.social） |
| `BLUESKY_APP_PASSWORD` | Bluesky で発行したアプリ専用パスワード |
